# 📊 HR Analytics Dashboard

## 📌 Project Overview
This project analyzes employee attrition using Microsoft Excel. The dashboard was built to help HR teams identify workforce trends and key factors contributing to employee turnover.

---

## 🛠️ Tools Used
- Microsoft Excel
- Power Query
- Pivot Tables
- Pivot Charts
- Slicers

---

## 📈 Key Performance Indicators (KPIs)
- Total Employees
- Employees Left
- Attrition Rate
- Average Monthly Income
- Average Age
- Male Employees
- Female Employees

---

## 📊 Dashboard Features
- Department-wise Attrition Analysis
- Age Group Analysis
- Gender Distribution
- Salary Slab Analysis
- Business Travel Analysis
- Interactive Slicers

---

## 💡 Key Insights
- Research & Development (R&D) had the highest employee attrition.
- Employees working overtime experienced higher attrition.
- Employees aged 26–35 had the highest turnover.
- Lower salary groups experienced higher attrition.
